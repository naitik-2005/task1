# Customer Review Analysis Project

This is a **Python project** for analyzing customer reviews and purchase data. It includes data cleaning, feature engineering, and categorization for better insights.

## Project Overview

The project uses Excel files containing customer data and reviews. The goal is to:

- Clean missing values in the dataset  
- Fill missing **Review Ratings** category-wise using median  
- Categorize customers into **age groups** (`Young Adult`, `Adult`, `Middle-Aged`, `Senior`)  
- Prepare the dataset for analysis and visualization  

---

## Files in the Repository

- `sales.ipynb` – Jupyter Notebook containing all code and analysis  
- `costomer_shopping.xlsx` – Original customer shopping data  
- `customer_cleaned_file.xlsx` – Cleaned dataset ready for analysis
